# 🔐 Gerador de Senhas e QR Codes

Este é um projeto desenvolvido em Node.js com o objetivo de gerar:

- 🔑 Senhas seguras
- 📱 QR Codes

## 📌 Funcionalidades

- Geração de senhas aleatórias e seguras
- Criação de QR Codes a partir de textos ou links

## 🚀 Tecnologias utilizadas

- Node.js
- JavaScript
